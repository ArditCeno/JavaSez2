# Sezoni2

Ky repository përmban të gjitha ushtrimet praktike të lëndës **Programim në Java** (Sezoni 2). 
Puna përfshin 6 laboratorë dhe 13 seminare që mbulojnë konceptet themelore dhe të avancuara të Java-s.

## Përmbajtja

| Dosja | Përshkrimi |
|-------|------------|
| `Lab1` - `Lab6` | Ushtrime laboratorike: variabla, kushte, loop-e, vargje, OOP, Swing GUI, file I/O |
| `Seminar1` - `Seminar13` | Ushtrime javore seminari: struktura të dhënash, algoritme, GUI, lojëra |
| `Leksione` | Shembuj nga leksionet (vizatime, roboti) |
| `PerseritjeArdit` | Ushtrime përsëritjeje |
| `PerseritjeTest` | Zgjidhje provimesh të kaluara (Tez 2024A/B, Tez 2025A/B) |
| `UML` | Sistemi i Menaxhimit të Provimeve (Swing me role login) |

## Karakteristikat Kryesore

- **Lojë Mario Platformer** (`Lab3/MarioGame`) — lojë 2D me grafikë Swing, platforma, armiq, monedha dhe topa zjarri
- **Sistemi i Menaxhimit të Provimeve** (`UML`) — aplikacion Swing me 3 tab-e: Studenti, Sekretari, Profesori
- **Rezervim Hoteli** (`Lab4`) — formë rezervimi me lloje dhomash dhe shërbime shtesë
- **Analizë Frekuence Fjalësh** (`Seminar13`) — lexon tekst dhe paraqit frekuencën e fjalëve
- **Menaxhim Pronash** (`Seminar8`) — aplikacion për menaxhimin e pronave me oferta klientësh

## Teknologjitë

- **Java** (JDK 25)
- **Java Swing** për ndërfaqen grafike
- **FlatLaf 3.7.1** për pamje moderne

## Si të ekzekutoni

1. Hapeni projektin në IntelliJ IDEA
2. Sigurohuni që JDK 25 është konfiguruar
3. Shtoni librarinë `flatlaf-3.7.1.jar` (gjendet në `libs/`)
4. Ekzekutoni klasën e dëshiruar nga çdo paketë
